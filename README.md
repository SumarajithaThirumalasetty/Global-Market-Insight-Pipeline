# Global-Market-Insight-Pipeline
The Global Market Insight Pipeline is an end-to-end data engineering project that collects global market data, processes it, analyzes trends, and publishes insights for business decision-making. This project demonstrates skills in ETL, cloud services, automation, and dashboarding
Data Sources (Stocks, Forex, Crypto, Commodities)
        ↓
Ingestion Layer (API / CSV / Web Scraping)
        ↓
Raw Data Storage (S3 / ADLS → raw zone)
        ↓
Processing Layer (Python / Pandas / PySpark)
        ↓
Curated Storage (Parquet / Tables)
        ↓
Analytics Layer (Athena / Redshift / Synapse)
        ↓
Dashboard (Power BI / QuickSight)
        ↓
Automation (Airflow / ADF / Step Functions)
